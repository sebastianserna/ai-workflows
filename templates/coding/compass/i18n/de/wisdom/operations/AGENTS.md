# Agents: operations/

> Wie Code in die Produktion gelangt und am Laufen bleibt.

## Was Sie hier finden

- CI/CD-Pipeline-Konfiguration
- Umgebungs-Setup und Variablen
- Bereitstellungsverfahren pro Umgebung
- Monitoring und Observability
- Beantwortet **„wie wird es bereitgestellt und betrieben"**

## Wann diesen Ordner konsultieren

- Bevor Sie Bereitstellungs- oder CI/CD-Konfiguration ändern
- Beim Hinzufügen von Umgebungsvariablen oder neuen Diensten
- Bei der Fehlerbehebung in der Produktion
