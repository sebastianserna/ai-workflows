## Работа с ИИ-агентами

Этот проект настроен для работы с ИИ-агентами, такими как Claude Code:

- **AGENTS.md** — Полное руководство для агентов по пониманию проекта
- **CLAUDE.md** — Точка входа, загружается автоматически при запуске Claude Code
- **wisdom/** — Техническая документация, с которой агенты сверяются при принятии решений
- **workplan/** — Система управления задачами, которую агенты могут читать и обновлять
