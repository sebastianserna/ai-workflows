# CLAUDE.md

**Конфигурация Claude Code для этого проекта.**

## Основные инструкции

Все соглашения, правила, структура проекта и руководства по стилю определены в:

@AGENTS.md

Прочитайте этот файл перед выполнением любой задачи.

## Ключевая структура

| Папка | Назначение |
|-------|------------|
| `wisdom/` | База знаний проекта (архитектура, руководства, решения) |
| `workplan/` | Управление рабочими планами (`(draft/) -> backlog/ -> coding/ -> done/`) |

## Планы и решения

Прежде чем предлагать улучшения, проверьте `workplan/` на наличие существующих планов, чтобы избежать дублирования.
