# CLAUDE.md

**이 프로젝트의 Claude Code 설정.**

## 주요 지침

모든 규칙, 프로젝트 구조 및 스타일 가이드는 다음에 정의되어 있습니다:

@AGENTS.md

작업을 수행하기 전에 해당 파일을 읽어주세요.

## 주요 구조

| 폴더 | 용도 |
|------|------|
| `wisdom/` | 프로젝트 지식 베이스 (아키텍처, 가이드, 의사결정) |
| `workplan/` | 작업 계획 관리 (`backlog/ -> coding/ -> done/`) |

## 계획 및 의사결정

개선을 제안하기 전에, `workplan/`에서 기존 계획을 확인하여 중복을 방지하세요.
