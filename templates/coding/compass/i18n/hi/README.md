## AI एजेंट के साथ काम करना

यह प्रोजेक्ट Claude Code जैसे AI एजेंट के साथ काम करने के लिए कॉन्फ़िगर किया गया है:

- **AGENTS.md** — एजेंट को प्रोजेक्ट समझने के लिए संपूर्ण गाइड
- **CLAUDE.md** — एंट्री पॉइंट जो Claude Code शुरू होने पर स्वचालित रूप से लोड होता है
- **wisdom/** — तकनीकी दस्तावेज़ीकरण जिसे एजेंट निर्णय लेने के लिए परामर्श करते हैं
- **workplan/** — कार्य प्रबंधन प्रणाली जिसे एजेंट पढ़ और अपडेट कर सकते हैं
